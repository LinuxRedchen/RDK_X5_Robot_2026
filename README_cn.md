# RDK_X5_Robot_2026
基于地平线RDK X5开发的智能移动机器人完整开源项目
## 🎯 项目功能
1. YOLO系列目标检测、视觉识别
2. 麦克纳姆轮底盘运动控制
3. Mjpg-streamer网页实时图传
4. ROS2 Humble远程通信控制
5. 激光雷达环境感知

## 📦 环境依赖
- 硬件：地平线RDK X5开发板
- 系统：Ubuntu 22.04
- 软件：Tengine、OpenCV 4.x、ROS2 Humble、Git
